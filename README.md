# SYSTEM-THINKING-AND-LOGIC-ASSIGNMENT
System Thinking & Logic Assignment

This repository contains our group assignment for the System Thinking & Logic subject.
The project focuses on identifying a real-world problem, analysing the system involved,
and proposing a logical solution using system thinking principles.

Contents:

- System Analysis
- Logic Flow & Explanation
- Proposed Solution
- Reflection
